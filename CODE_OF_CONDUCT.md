# Code of Conduct

This project is committed to a respectful, constructive, and inclusive
community.

## Expected behavior

- Be respectful in language and feedback.
- Assume good intent, but communicate clearly and directly.
- Focus on ideas, bugs, and code rather than people.
- Accept maintainers' decisions about scope, quality, and release timing.

## Unacceptable behavior

- Harassment, threats, or personal attacks.
- Discriminatory or hateful language.
- Bad-faith disruption, trolling, or spam.
- Publishing private information without consent.

## Reporting

Do not use public issues for sensitive conduct reports.

Report concerns privately to the repository owner through GitHub or, if
enabled, through GitHub's reporting or moderation tools.

## Enforcement

Maintainers may remove comments, close threads, or block contributors whose
behavior harms the project or its community.
